## 🎯 What we focus on?!

우선, NC2의 목표인 '애플 생태계의 기술을 탐구해보자'를 잘 이루기 위해서 ML에 대해서 알아갈 수 있었다.
<br> 1주안에 모델이 학습해야하기 때문에, 많은 raw데이터들을 필요로 해서 6가지 모델 중 Image와 Motion, Sound를 선택했다. 
<br> 왜냐하면, 우리가 해보고 싶고 1주라는 짧은 시간안에 도전해볼만한 기술을 선택했기 때문이다.
